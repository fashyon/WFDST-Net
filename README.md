# WFDST-Net
The code of the paper "Wavelet Frequency Division Self-attention Transformer Image Deraining Network"<br>
论文《小波分频自注意力Transformer图像去雨网络》的代码

#Important Requirements
python = 3.7
pytorch = 1.9
opencv-python = 4.5
numpy = 1.21

#Getting Started
##Training
Run shell scripts to train the models
`python train.py`

##Testing
`python show.py`

##Evaluation metrics
`python eval.py`
